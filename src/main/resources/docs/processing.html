Automatic Processing

Since version 1.4, the annotation can be supported by automatic tools. They are accessible via the Tools menu and the list will be extended in the future.
Important: Some components only work for specific languages. Before running them, you will need to set the document language correctly. Please also note that many components do not work flawlessly. They are included to speed up the annotation process, but do not replace manual inspection.

The supported components are:

Stanford NER

The Stanford NER system is run over the entire document. All proper names with the same surface string are then turned into mentions of an entity.
