Queues A command queue can be in three states:
check_circle Enabled
The queue accepts and releases commands.
highlight_offBlocked
The queue accepts commands but does not release them. These commands may be released manually, or automatically as soon as the queue moves to the enabled state.
lensDisabled
The queue does not accept commands. Any attempt to issue a command linked to a disabled queue will fail immediately.
